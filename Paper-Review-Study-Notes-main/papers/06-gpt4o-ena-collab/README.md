## Problem
- 협업 역량(collaborative competency)은 학습 성과의 핵심 요소이지만,
  정답 정확도와 같은 개인 단위 지표만으로는 충분히 평가하기 어렵다.
- 기존 협업 분석 연구는 주로 수작업 주석이나
  도메인 특화 모델에 의존해 확장성과 비용 측면에서 한계를 가진다.

## Key Idea
- GPT-4o를 **fine-tuning 없이 prompting만으로** 활용하여
  협업 역량을 자동 주석하는 시스템(CoComTag)을 제안한다.
- Epistemic Network Analysis(ENA)를 통해
  LLM 주석과 인간 주석 간의 구조적 차이를 분석한다.

## Why I Read This
- LLM이 단순 분류 성능을 넘어서
  협업과 같은 복합적인 사회적 역량을 어디까지 포착할 수 있는지 궁금했다.
- LLM + ENA 결합 방식이
  교육·CSCL 연구에서 어떻게 활용될 수 있는지 알고 싶었다.

## One-line Takeaway
> 이 논문은 **GPT-4o가 prompting만으로 협업 역량을 상당 수준 포착할 수 있지만,
> 인간과 동일한 방식으로 협업 구조를 이해하지는 않음을 보여준다.**
